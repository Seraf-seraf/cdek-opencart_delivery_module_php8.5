# CDEK Delivery Shipping для OpenCart на PHP 8.5

Это форк официального модуля доставки CDEK для OpenCart, адаптированный для работы на PHP 8.5.

Модуль добавляет расчёт стоимости доставки через API CDEK, выбор пункта выдачи и работу с отправлениями из административной панели OpenCart.

## Требования

- PHP 8.5;
- расширение PHP cURL;
- OpenCart 3.x.

## Установка

1. Скачайте готовый архив `opencart-cdek-delivery-php8.5.ocmod.zip`.
2. В административной панели OpenCart откройте **Расширения → Установка расширений** и загрузите архив.
3. Откройте **Расширения → Модификаторы** и обновите кеш модификаторов.
4. Перейдите в **Расширения → Расширения**, выберите тип **Доставка** и установите модуль CDEK.
5. Откройте настройки модуля и укажите данные для подключения к API CDEK.

Архив уже содержит Composer-зависимости в каталоге `vendor`, поэтому выполнять `composer install` на сервере не требуется.

Перед установкой на рабочий магазин рекомендуется создать резервную копию файлов и базы данных.

## Сборка установочного архива

Установите production-зависимости:

```bash
composer --working-dir=upload/system/library/cdek_official install \
  --no-dev \
  --optimize-autoloader
```

Затем из корня репозитория выполните:

```bash
zip -r opencart-cdek-delivery-php8.5.ocmod.zip install.xml upload \
  -x '*.git*' \
  -x '*/.idea/*'
```

В корне готового архива должны находиться `install.xml` и каталог `upload`.

## Происхождение

Проект является независимым форком официального модуля CDEK. Основное назначение форка — совместимость с PHP 8.5. Названия и товарные знаки CDEK принадлежат их правообладателям.
